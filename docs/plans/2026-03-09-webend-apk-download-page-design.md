# Webend APK 下载页设计

**日期**: 2026-03-09
**项目**: 早护通 APK 下载页
**类型**: 静态网页

---

## 1. 项目概述

为早护通 App 创建一个现代风格的下载落地页，用户点击下载按钮后播放手绘水彩风格动画，然后开始下载 APK。

## 2. 功能需求

### 2.1 动态下载配置
- 读取 `public/downloads/` 目录
- 支持两种配置方式：
  - 放置 `link.txt` 文件：内容为外部 CDN 链接
  - 直接放置 APK 文件：自动识别并下载

### 2.2 页面内容
- App 名称：早护通
- 版本号：自动读取配置目录或硬编码
- 简介：记录早产儿生长曲线，提供医疗 AI 助手，早产儿呵护课堂
- 下载按钮

---

## 3. 视觉设计

### 3.1 色彩方案
| 用途 | 颜色 | Hex |
|------|------|-----|
| 背景色 | 暖米色 | #FDF6E3 |
| 主色调 | 暖黄色 | #E8B86D |
| 文字色 | 深棕色 | #5D4E37 |
| 辅助色 | 柔和粉 | #F5E6D3 |

### 3.2 插画风格
- **手绘线稿** + **水彩晕染**
- 柔和、温暖、富有故事性

---

## 4. 动画设计

### 4.1 动画流程
1. **初始状态**：显示静态插画（母亲抱着婴儿）
2. **点击下载**：
   - 母亲开始轻微摇晃（手绘逐帧感）
   - 婴儿身上出现柔和光芒
   - 光芒变成星星向外散开（水彩晕染效果）
   - 下载开始
3. **下载中**：按钮变成进度条
4. **下载完成**：星星消散，恢复平静

### 4.2 技术实现
- 使用 Lottie 动画或 CSS 关键帧动画
- 手绘风格通过 SVG 贴图实现
- 水彩效果通过 CSS filter + 模糊实现

---

## 5. 技术栈

- Vue 3 + Vite
- TypeScript
- 纯 CSS 动画（不使用重型动画库）
