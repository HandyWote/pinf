# 早护通 APK 下载页美化设计方案

## 1. 项目概述

- **项目名称**: 早护通 APK 下载落地页
- **项目类型**: 静态展示页面 (Vue 3 + Vite)
- **核心功能**: 展示 App 信息，提供 APK 下载
- **目标用户**: 需要下载早护通 App 的用户（新生儿/早产儿家长）

## 2. 视觉与交互设计

### 风格定位
- **整体风格**: 手绘温馨风 + 现代3D视差效果
- **色彩方案**: 温暖柔和的奶油色调 + 柔和水彩色块点缀
- **字体**: 手写风格标题 + 现代无衬线正文

### 核心视觉元素
1. **3D悬浮卡片**
   - 放置手绘风格的母亲抱婴儿插画（PNG透明背景）
   - 卡片具有 3D 倾斜效果，跟随鼠标移动
   - 轻微的悬浮阴影，增强立体感

2. **背景设计**
   - 柔和的奶油色渐变背景
   - 散落的水彩风格装饰元素（圆点、花朵等）
   - 轻微的噪点纹理增加质感

3. **动画效果**
   - 页面加载：卡片从下方浮入 + 淡入
   - 鼠标移动：卡片 3D 倾斜（max 15度）
   - 滚动页面：卡片有视差位移
   - 点击下载：卡片按下效果 + 星星/爱心爆发动画
   - 悬停按钮：微微放大 + 发光

### 组件清单
1. **3D 卡片组件** - 承载手绘图片，支持 3D 倾斜
2. **下载按钮** - 带动画的 CTA 按钮
3. **装饰元素** - 水彩风格背景装饰
4. **信息区域** - App 名称、版本号、描述

## 3. 技术方案

### 技术栈
- Vue 3 + TypeScript
- Vite 构建
- 纯 CSS 3D Transform 实现倾斜效果（无需 Three.js）

### 实现要点
- 使用 `transform: rotateX() rotateY()` 实现 3D 倾斜
- 监听 `mousemove` 计算鼠标位置映射到旋转角度
- 使用 `requestAnimationFrame` 保证流畅动画
- 点击下载时添加粒子爆发效果（星星/爱心）

## 4. 验收标准

- [ ] 页面加载有流畅的入场动画
- [ ] 鼠标移动时卡片有自然的 3D 倾斜效果
- [ ] 点击下载按钮有明显的视觉反馈
- [ ] 整体风格温馨、手绘感强
- [ ] 移动端有适配（可以简化 3D 效果）
- [ ] 下载功能正常工作
